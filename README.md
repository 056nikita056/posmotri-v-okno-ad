# Проект «Посмотри в окно»

Это учебный проект, где нужно дописать стили для уже готового приложения.  
HTML и JavaScript менять нельзя, требуется только оформить страницу в style.css по макету

## Что делает приложение

- Показывает большое видео слева.
- Справа список карточек с другими городами.
- Снизу форма поиска по городу и времени суток
- Есть прелоадеры при загрузке и сообщение об ошибке.

## Что нужно сделать

- Собрать лейаут: видео слева, список карточек справа, форма снизу.
- Задать точные размеры блоков.
- Настроить список карточек со скроллом.
- Оформить карточки, текст и превью по макету.
- Сверстать кастомные чекбоксы и поле ввода.
- Настроить кнопки «Найти» и «Показать ещё» и их состояния.
- Сделать все состояния элементов: hover, active, focus-visible.
- Сделать выделение активной карточки.
- Правильно расположить прелоадеры (только внутри своих блоков)

## Файлы проекта

- `index.html` — готовая разметка.
- `style.css` — здесь нужно написать все стили.
- `script.js` — логика работы (не трогаем).
- `fonts/`, `images/` — ресурсы проекта.

## Как проверить

Открыть `index.html` в браузере и сверить с макетом:  
всё должно выглядеть и работать так же как в дизайне
